# 机构抱团与踩踏

## 一、概述

机构抱团是A股大市值股票层面最核心的博弈形态。其驱动力来自中国公募基金的**相对排名考核制度**，而非绝对收益。当大量基金经理的持仓高度重叠时，价格运动被放大为两部分：基本面驱动的合理上涨 + 抱团导致的正反馈加速。正反馈的反面是踩踏——当抱团松动时，同样的结构会以比上涨更快的速度坍塌。

对 ETF 轮动策略而言，理解抱团与踩踏的机制，直接关系到：你在追的动量信号是基本面信号还是抱团自强化信号？如果是后者，反转的时机和幅度是什么？

---

## 二、抱团的制度根源：相对排名考核

### 2.1 公募基金的核心激励机制

中国公募基金经理的KPI是**相对排名**（相对基准或相对同类产品），而非绝对收益。这产生几个关键行为模式：

1. **排名压力在期末集中**：年度排名考核驱动四季度（尤其是11-12月）的异常调仓行为
2. **相对排名 → 跟踪误差厌恶**：跑输同行的惩罚远大于跑输基准。结果是——如果整个行业都在买某只票，你不买就是自己找死（相对排名落后）
3. **相对排名 → 不怕一起错**：如果大家一起亏，你的排名不变（相对位置不变）。但如果大家都亏你一个人亏更少，排名反而提升

### 2.2 这如何催生抱团

假设板块A的基本面开始改善。第一批买入的基金获得了超额收益。第二批基金看到A板块涨了、自己的配置里没有A，面临客诉和排名压力→被迫追买。第三批看到第一第二批都买了→不买就落后→继续追。

这不是「同谋」或「信息共享」，而是在相对排名考核下每个人的独立理性选择聚合成集体非理性行为。经济学家称之为**委托人-代理人问题**在基金行业的具体表现。

## 三、抱团的正反馈机制

### 3.1 价格-资金流循环

```
基金买入 → 股价上涨 → 净值上升 → 排名提升 
    → 吸引新申购资金 → 用新资金继续买入 → 股价继续上涨
    → 排名继续提升 → 更多申购资金 → ...
```

这是一个正反馈循环。关键环节：

- **申购资金放大**：基金净值在同类中名列前茅 → 零售渠道（银行、蚂蚁、天天基金）将其排在前列 → 散户跟风申购 → 基金经理被迫用新钱继续买 → 进一步推高持仓股
- **没有「止盈」的概念**：赚了→排名上升→继续买（直到排名压力消失）。不存在类似游资的「涨到位了就出货」，因为「出货」=主动降低排名

### 3.2 抱团如何扭曲价格信号

在抱团驱动的上涨中，价格的上涨不再完全反映信息（基本面改善），而是包含了一个自反馈分量：

```
实际价格 = 基本面公允估值 + 抱团溢价（自反馈分量）
```

抱团溢价是脆弱的——它的存在依赖于申购资金的持续流入。一旦资金流逆转，这部分溢价会迅速消失，且消失的速度通常远快于积累的速度。

## 四、抱团何时崩溃：触发踩踏的机制

### 4.1 核心问题

你问的是「他们什么时候会dump position」。答案是：**不是「他们决定dump position」，而是「外部约束迫使他们dump position」。** 基金经理个体没有主动砸盘的自由度——他们被申购赎回机制和排名压力所约束。

### 4.2 触发机制一：基本面预期逆转（最根本）

当抱团标的的基本面出现恶化的证据（业绩不及预期、行业政策转向、竞争格局恶化等），一部分最先发现信号的基金会开始减仓。

但问题在于：**在抱团中，「最早减仓」本身就是一件很难做的事。**

理性的基金经理会想：
- 如果我减仓了、其他人没减、股价没跌→我排名下降→我错了
- 如果我减仓了、其他人也减了、股价跌了→排名不变→对排名没帮助
- 如果我不减仓、别人减了、股价跌了→我亏同样多→排名不变

所以基本面上「应该减仓」的判断，在相对排名考核下不构成充分的减仓理由。这导致基本面恶化信号需要积累到临界点之上——到了**不跑不行**的时候——才会触发集体行动。

### 4.3 触发机制二：增量资金枯竭（最常见）

正反馈循环的核心燃料是申购资金。当：

- 市场整体风险偏好下降（熊市预期）
- 银行存款搬家趋势逆转
- 基金限购（规模过大影响操作）
- 监管窗口指导（提示泡沫风险）

任一条件导致申购资金增速放缓或转为净赎回，循环断裂。

**关键机制**：申购资金减速 → 基金经理无法继续加仓 → 股价上涨动力减弱 → 排名从头部退到中部 → 零售渠道不再推荐 → 申购进一步减速 → 转为净赎回 → 被迫卖出 → 股价下跌 → 净值下跌 → 更多赎回。

这是和上涨相反的负反馈循环——一旦启动，自我强化。

### 4.4 触发机制三：季度末/年末的「排名保卫战」（最规律）

这是可量化程度最高的踩踏触发点。

假设到11月底，基金经理A排名第15（前10%）。他的持仓中包含一只已经涨了80%的票。他的决策：

**情景A：继续持有到12月底**
- 如果该票在12月继续涨→排名更高
- 如果该票在12月回调→可能被第16-20名反超→排名跌出前10%

**情景B：12月初减仓，锁定浮盈**
- 如果该票继续涨→他排名下降（因为没满仓）→但下跌风险已经消除
- 如果该票回调→他的仓位比其他人轻→排名反而上升

在不确定性面前，排名靠前的人有强烈的「落袋为安」动机。当多数排名靠前的基金同步减仓同一批抱团股时，踩踏发生。

**季度末（3/6/9/12月）的规律性调仓**是ETF轮动策略可以捕获的规律性信号。

### 4.5 触发机制四：外部冲击下的赎回潮

市场出现系统性利空（如2022年4月上海封控引发的A股暴跌、2015年股灾）→基金净值暴跌→散户恐慌赎回→基金经理被迫不计价格卖出→进一步压低净值→更多赎回。这是经典的「赎回-卖出-净值下跌」负反馈螺旋。

参见 [[2015年股灾始末]]。

## 五、抱团与踩踏的识别

### 5.1 事后识别（已发生的抱团）

| 指标 | 抱团信号 |
|------|---------|
| 基金重仓股重合度 | 前十大重仓中，某几只股票出现在>60%同类基金中 |
| 重仓股集中度 | 同一基金多只重仓股属同一板块/主题 |
| 板块换手率异常上升 | 板块日均换手率大幅超过历史均值 |
| 基金持仓报告 | 季报中的前十大重仓集中度趋势上升 |
| 板块相对强弱 | 该板块持续跑赢大盘，且跑赢幅度加速 |

### 5.2 前瞻识别（即将松动的信号）

| 指标 | 松动信号 |
|------|---------|
| 持仓集中度见顶回落 | 季报中同一板块的持仓比例开始下降 |
| 板块换手率在上涨中暴增 | 高位放量换手=有人在出 |
| 龙头股率先回调 | 抱团板块内龙头票走弱而跟风票还在涨（内在强度分化） |
| 基金申购数据 | 相关主题基金从净申购转净赎回 |
| 季度末异常波动 | 3/6/9/12月最后两周，抱团股出现与基本面无关的异常波动 |

## 六、踩踏为何比上涨更快

物理原因很简单：

**买入时**：用现金买入股票。现金的供给几乎是无限的（理论上可以一直募资），所以买入推动价格上涨的速度受制于卖方的供给。

**卖出时**：需要找到买方。买方的资金不是无限的，尤其在恐慌时，所有人都在卖、没有人愿意接。卖出压低的幅度取决于买方订单簿的深度。

还记得 [[砸盘博弈]] 中的边际定价吗？一批基金同时卖出→买方订单簿被打穿→价格远低于基本面→低净值触发更多赎回→更多卖出→恶性循环。

**不对称性的根源**：买入是「资金吸纳」过程（可以有源源不断的新资金），卖出是「资金挤出」过程（买方有限）。

## 七、对 ETF 轮动策略的启示

### 7.1 动量信号中的抱团分量

如果你的ETF轮动策略依赖动量因子（涨得好的继续涨），你在一定程度上是在交易抱团——因为在大市值板块中，强动量往往包含抱团自反馈分量。

关键问题：你的动量信号中，多少是基本面驱动的可持续趋势、多少是资金流驱动的脆弱的自反馈？

一个可行的区分思路：
- 考察板块上涨过程中，**龙头股内部是否出现分化**。龙头持续走强+跟风股跟上=健康动量。龙头走弱+跟风股单独走强=抱团末期（先行者已在撤）。
- 考察**板块上涨与资金流入的关系**。被动ETF资金流入（市场性）vs 主动基金重仓增持（机构性）→后者更容易形成抱团。

### 7.2 季度末效应

ETF轮动可以在3/6/9/12月最后1-2周调高对抱团瓦解风险的权重。这不是说「季度末必跌」，而是说季度末发生踩踏的条件概率升高——你可以把它作为一个交互特征（而非主因子）。

### 7.3 赎回潮的预警指标

关注：
- 货币基金收益率 vs 股市预期收益率（资金在搬家吗？）
- 新发基金募集规模（增量资金的速度）
- 相关ETF的折溢价变化（市场情绪的实时温度计）
- 融资余额变化（杠杆资金的参与度）

这些不是直接信号，而是环境变量——它们告诉你当前市场处于抱团周期的哪个阶段。
