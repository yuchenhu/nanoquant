# 股指期货基差与期现联动

> 创建日期：2026-07-06

---

## 一、概述

股指期货是 A 股最重要的衍生工具，基差（期货-现货的差额）包含了丰富的市场结构信息。在知识网络中，基差数据直接嵌入[[ETF动量质量因子]]中"衍生品基差"维度，基差的形成机制（中性策略做空→贴水）与[[融资融券与两融数据]]中的强平螺旋共享"持仓结构→价格压力"的分析范式。期货市场的对手盘结构（中性策略、套利者、投机者）也与[[机构抱团与踩踏]]和[[私募行业结构与α容量诅咒]]形成连接。

散户直接交易期货的投入产出比极低，但基差数据作为 ETF 轮动的辅助信号来源价值很高。

---

## 二、核心机制

### 2.1 市场结构与合约体系

中金所（CFFEX）目前有四个股指期货品种：

| 品种 | 标的指数 | 乘数 | 合约面值（约） | 上市日期 |
|------|---------|:---:|:------------:|---------|
| IF | 沪深300 | 300 | 144万 | 2010-04-16 |
| IH | 上证50 | 300 | 88万 | 2015-04-16 |
| IC | 中证500 | 200 | 172万 | 2015-04-16 |
| IM | 中证1000 | 200 | 169万 | 2022-07-22 |

关键规则：
- 合约月份：当月、下月、当季（3/6/9/12）、下季
- 最后交易日：合约到期月份第三个周五
- 交割方式：**现金交割**（到期按指数结算价现金差额结算）
- 保证金：约 12%（交易所标准），期货公司加收后约 15%
- 实际杠杆：约 8.5x（vs 两融的 2.25x）

### 2.2 基差定义

```
基差（Basis）= 期货价 - 现货价

贴水（期货 < 现货）：基差为负，A股2015年后的常态
升水（期货 > 现货）：基差为正，2015年前常见，现在罕见
```

年化基差需考虑距到期天数：
```
年化基差 = (基差/现货) / 距到期天数 × 365
```

### 2.3 贴水的三层驱动力量

**第一层：量化中性策略的结构性空头（主因）**

```
中性策略操作：
  做多一篮子精选股票 + 做空等市值的 IC/IM 期货
  = 净头寸市场中性（对冲 beta，保留选股 alpha）

结果：
  中性策略永远在卖出期货（不挑价格、市价做空）
  → 期货卖方长期压倒买方
  → 期货被结构性地压在现货以下 → 贴水
```

**第二层：2015年股灾后散户多头的永久性消失**

2015年前散户在 IF 上做多投机活跃，期货升水是常态。股灾后中金所大幅限制开仓、提高保证金、暴增平今仓手续费——散户多头群体被摧毁后至今未恢复。

**第三层：雪球产品的双向效应**

雪球发行方（券商自营）为了对冲卖出的 put，做多 IC/IM 期货。但方向是反直觉的：
- 温和上涨时：delta 下降 → 减少多头 → **卖出**期货 → 加剧贴水
- 急速下跌时：delta 急升 → 增加多头 → **买入**期货 → 缓解贴水

### 2.4 贴水不是"看空信号"

A 股常态贴水 ≠ 市场看空。它反映的是**结构性对冲需求**，而非方向判断。只有当贴水远深于正常水平时（如超过均值 2 个标准差），才是有意义的异常信号。

## 三、期现套利与跨期套利

### 3.1 期现套利（期货 vs 现货）

**核心逻辑**：到期日期货必须等于现货（现金交割保证），在此之前任何偏离都可以套利。

```
期货贴水异常深 → 买入廉价期货 + 卖空现货（ETF/成分股）
  → 到期时两者收敛 → 赚贴水回归的钱
  → 方向风险被对冲，净赚收敛收益
```

**A 股的实际瓶颈**：融券做空 ETF 困难（券源受限），标准套利常无法执行。替代方案：
- 用已有持仓卖出现货，换入期货（存量切换）
- 买方天然选择期货而非 ETF（增量分配）

### 3.2 跨期套利（不同到期月的合约之间）

**原理**：两个合约最终收敛到同一个现货价，但受不同强度的对冲压力。套利原则是**买便宜、卖贵**。

```
A股实际操作：买入远月（被中性策略集中做空，太便宜）+ 卖出近月

建仓时价差异常宽 → 两腿方向相反 → 现货涨跌互相抵消
→ 净赚"价差回归"的钱
```

**操作细节**：
- 近月到期当天，远月也同步平仓——不会让远月裸奔
- 中金所对跨期套利持仓有保证金优惠（约正常水平的 20-30%）
- 选流动性最好的合约组合（当月+当季，如 IC2607+IC2609）

### 3.3 年化贴水的均衡水平

```
年化贴水 ≈ 套利成本 + 套利不完美的摩擦溢价

套利成本 = 融券成本（2-3%）+ 资金成本（2%）≈ 5%
摩擦溢价 = 融券不可得造成的套利无法执行 → 约 5%
实际贴水 ≈ 10%（IC/IM 实测值）

美股因做空便利，摩擦溢价≈0，贴水≈纯套利成本。
```

## 四、期货曲线结构

不同到期月合约的实际价格结构（2026-07-06，中证500）：

```
价格
8651 ─ ● 现货
8625 ─ ─ ● IC2607（当月，11天）  贴水 -0.30%  年化 -10.1%
8538 ─ ─ ─ ● IC2608（次月，46天）   贴水 -1.31%  年化 -10.4%
8467 ─ ─ ─ ─ ● IC2609（当季，74天）  贴水 -2.13%  年化 -10.6%
```

**关键规律**：
- 远月贴水比近月更深——对冲空头集中在季度合约
- 年化贴水率在所有合约上几乎一致（~10%）——跨期套利保证了定价一致性
- 当季合约（IC2609）流动性最好（日成交 12 万张），是中性策略的主要做空目标

## 五、对手盘结构

| 角色 | 方向 | 获利来源 | 量级 |
|------|:---:|---------|------|
| 量化中性策略 | 空头 | 选股 alpha - 贴水成本 | 主力 |
| 投机多头 | 多头 | 指数方向（赌涨） | 散户+机构 |
| 期现套利者 | 多头（期货）+ 空头（现货） | 贴水收敛的价差 | 中等 |
| 跨期套利者 | 买远月 + 卖近月 | 跨期价差回归 | 中小 |
| ETF 做市商 | 双向 | 买卖价差 | 流动性提供 |
| 雪球发行方 | 多头（对冲 put） | 对冲需求，非盈利 | 中等 |

**套利者是贴水回归的根本力量**——贴水越深，套利利润越大，参与套利的资金越多，贴水被拉回均衡的速度越快。

## 六、基差数据的量化应用

### 6.1 可构造的信号

| 信号 | 构造方法 | 含义 |
|------|---------|------|
| 基差水平 | `(fut - spot) / spot` | 绝对贴水深度 |
| 基差变化率 | `Δbasis / |basis_avg|` | 贴水快速收窄=空头撤退=看涨 |
| 基差结构 | `近月基差 - 远月基差` | 曲线陡峭程度反映对冲需求的期限分布 |
| 基差异常 | `|basis - MA20| > 2σ` | 极端贴水才是信号，正常贴水是噪声 |
| 大小盘风格 | `IH基差 - IC基差` | 正=偏好大盘，负=偏好小盘 |

### 6.2 对 ETF 动量质量因子的嵌入

在[[ETF动量质量因子]]的"衍生品基差"维度中：

1. **基差方向**本身信息量有限（常态贴水不是信号）
2. **基差变化率**是高质量信号——贴水从深度中快速收窄往往伴随市场转暖
3. **基差异常**（超过 2σ 的贴水扩大）是可靠的负面预警
4. **期货升水**（罕见）是强烈的看多信号——市场愿意为期货支付溢价
5. 基差变化会通过[[基金净值与申赎机制]]传导至 ETF 折溢价和申赎流

### 6.3 期货直接替代 ETF 持仓的前提

贴水市场中做多期货比持有 ETF 有天然优势（展期收益约 7-10%/年 + 免管理费），但前提苛刻：
- 资金量 200 万+（1 张 IF 面值 144 万）
- 自动化的保证金管理和展期脚本
- 能承受单月 10% 回撤
- 不适用于当前 ETF 轮动策略阶段
