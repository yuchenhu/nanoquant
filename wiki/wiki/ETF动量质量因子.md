# ETF动量质量因子

> 创建日期：2026-07-02 ｜ 来源：讨论ETF动量效应的多维分解方法

---

## 一、概述

A股个股层面原始动量（1-12月窗口）以反转为特征，但通过区分交易结构和风险状态可以提炼动量信号（小单动量/大单反转、UMR框架）。ETF层面有独特的数据维度（双层市场、成分截面），可以用来分解动量的"质量"——动量相同但质量不同的ETF，未来表现有显著差异。

---

## 二、A股个股动量基础

### 2.1 原始动量的特征

- A股个股的Jegadeesh-Titman动量（过去3-12个月）长期不显著甚至为负——主要呈现反转。
- 原因是A股散户驱动的过度反应 + 公募短期考核导致的反向操作。

### 2.2 小单动量 / 大单反转

- **实证方向**：小单占比高的交易日 → 动量；大单占比高的交易日 → 反转。
- **机制**：小单无市场冲击，价格变动是真实供需反映→持续；大单推动导致过冲→回踩。机构净卖出不完全等于看空（大量是止盈行为）。
- **方正金工发现**：A股天然呈现"小单净买入、大单净卖出"的特征，牛市里加剧。

### 2.3 UMR（Unified Momentum and Reversal）框架

东方证券提出的动量反转统一框架。核心思路：不是二分法（今天算动量/反转），而是用每个交易日的风险状态做**连续加权**：

- **低风险日**（波动低、换手低）→ 正权重 → 提取动量成分
- **高风险日**（波动高、换手高）→ 负权重 → 提取反转成分

加权后的累积信号在1个月到1年的窗口下选股方向完全一致（都是动量），IC胜率94%+。比硬阈值过滤更稳健。

## 三、ETF独有的动量质量维度

ETF与个股最大区别：**一篮子资产 + 双层市场（一级申赎 + 二级交易）**。以下六个维度的原始数据均可通过Tushare的`index_weight`（月频成分股权重）+ `daily`（日频个股行情）间接获取，无需ETF日持仓PCF——A股被动ETF跟踪偏离度≤0.2%，误差完全可接受。

### 3.1 申赎资金流 × 价格动量

**数据源**：Tushare `fund_daily` 或 `fund_adj` 接口获取ETF份额（日频）

| 信号 | 含义 |
|------|------|
| 涨 + 净申购 | 资金驱动动量，正反馈，持续性最强 |
| 涨 + 净赎回 | 套利盘/国家队在卖，涨幅"内生动量"弱，反转风险高 |
| 连续净申购 + 连涨 | 最强动量确认 |
| 连跌 + 连续净赎回 | 恐慌踩踏，下行动量 |

**因子构造思路**：过去N日累计申赎净额（经AUM标准化）与同期累计收益的**同向一致性比例**（同时为正/负的天数占比）。一致性越高→动量越干净。

### 3.2 折溢价方向

**数据源**：Tushare `fund_daily` 或 `fund_nav` 接口获取折溢价率。`折溢价率 = 收盘价 / 单位净值 - 1`

| 信号 | 含义 |
|------|------|
| 溢价 + 涨 | 二级市场主动买盘强劲，做市商套利能力被压制，动量更可靠 |
| 折价 + 涨 | IOPV在涨但买盘不积极→被动跟随，缺乏主动做多力量 |
| 折价翻溢价 + 涨 | 早期趋势启动信号——AP开始申购套利，净申购即将来临 |

### 3.3 成分截面离散度（ETF核心维度）

**数据源**：`index_weight`（月频权重） + `daily`（日频个股收益）

**技术路径**：
1. 建立 ETF代码 → 跟踪指数代码 映射表
2. `pro.index_weight(index_code='指数代码', start_date='YYYYMMDD', end_date='YYYYMMDD')` → 成分股及权重
3. `pro.daily(ts_code=成分股列表, start_date=..., end_date=...)` → 日频收益
4. 每日用最新月频权重加权计算截面统计量

三个子指标：

**A. 收益离散度**：成分股日收益的截面标准差（等权或市值加权均可）。离散度高=少数龙头在拉，动量"脆弱"；离散度低=普涨，动量"扎实"。

**B. 上涨广度（Breadth）**：当日上涨成分数 / 总成分数。广度+动量同升→最干净动量信号；动量上升但广度下降→动量衰竭的早期预警。

**C. 头部集中度**：前5大权重股的加权收益贡献 / 全成分加权总收益。如果>50%，意味着5只股贡献了过半日变动→跟风盘不足，动量偏脆弱。

**三指标综合**：低离散度 + 高广度 + 低集中度 → 动量质量最高档。

### 3.4 量价协同（ETF上比个股干净）

ETF成交不存在对倒/拆单伪装污染。

| 信号 | 含义 |
|------|------|
| 放量上涨 | 真实资金流入（ETF成交与申赎高度相关），可信度高 |
| 缩量上涨 + 净申购继续 | 惜售，短期看涨 |
| 缩量上涨 + 净申购停止 | 动能耗尽的警报 |

**关键区分器不是量本身，而是量×申赎的交叉验证。**

### 3.5 尾盘成交占比

A股尾盘（14:50-15:00）ETF成交异常放大+价格上涨→往往是机构or国家队集中申购（汇金入市多选尾盘），有较强持续性。尾盘异常缩量→市场观望，动量可信度打折扣。

### 3.6 衍生品关联（部分ETF适用）

沪深300ETF对应IF期货、中证500ETF对应IC。

- 基差走强（贴水收窄/转升水）+ ETF上涨 → 衍生品确认现货动量
- 基差走弱 + ETF上涨 → 衍生品对持续性存疑

## 四、因子构造通用框架

参考UMR思路，不要做硬阈值过滤（"只取广度>0.7"），而是做连续性加权：

1. 对每个质量维度单独跑IC验证
2. 确认有效的维度做复合加权（等权或IC加权）
3. 用复合后的动量质量评分排序ETF

## 五、Tushare数据获取速查

| 数据 | Tushare接口 | 频率 |
|------|------|:---:|
| ETF基础信息（代码、名称、跟踪指数） | `fund_basic` | 日频更新 |
| ETF日行情（净值、折溢价） | `fund_nav` 或 `fund_daily` | 日频 |
| ETF份额（用于算申赎流） | `fund_adj` 或 `fund_daily` | 日频 |
| 指数成分股权重 | `index_weight` | **月频** |
| 成分股日行情 | `daily` | 日频 |
| 股指期货行情（用于基差） | `fut_daily` | 日频 |

**关键链条**：ETF → 跟踪指数 → `index_weight`（月频权重） + `daily`（日频收益） → 日频截面离散度/广度/集中度。ETF日均跟踪偏离度≤0.2%保证了指数近似的精度完全够用。
