# 识别市场主线

> 关联页面：[[机构抱团与踩踏]] [[ETF动量质量因子]] [[筹码结构与因子构造]] [[打板策略]] [[砸盘博弈]]

## 一、概述

**市场主线**是A股在某段时期内（通常3个月以上）被资金集中定价的核心方向——它决定了大盘的结构性涨跌，决定了ETF轮动策略的超额收益来源。

本文的核心观点：识别主线的关键不在于某个单一指标，而是**定性与定量两条腿的协同**：
- **定性**：回答「为什么这个方向值得定价」，解决逻辑真假问题——政策、产业趋势、盈利周期
- **定量**：回答「市场是否真的在定价这个方向」，解决时机与确认问题——资金、量价、扩散度

只定性不定量 → 逻辑正确但不知道市场是否已经反映、何时入场。只定量不定性 → 知道什么在涨但不知道是主线还是一日游。

识别主线的最终目的：在ETF轮动策略中，**给当前正在运行的板块赋予不同的置信度权重**，而非简单地追涨幅榜。

## 二、主线的定义与判定标准

### 2.1 主线 vs 轮动的本质区别

这里说的「主线」不是短期热点，而是具备以下四个维度共振的方向：

| 维度 | 主线 | 轮动/短期热点 |
|------|------|--------------|
| 持续时间 | 3个月+，可跨年 | 1-4周 |
| 逻辑深度 | 政策+产业+业绩三重共振 | 事件/情绪/季节性驱动 |
| 资金特征 | 持续净流入，占全市场成交15-30%，机构+游资合力 | 脉冲式流入，游资为主，散户跟风 |
| 扩散广度 | 从龙头→产业链→全板块，形成梯队 | 个股或小板块孤立联动 |

主线≠「涨得最多的板块」。主线是**有持续定价逻辑+有持续资金共识**的方向。

典型的主线案例：
- 2019年半导体（国产替代+5G周期）
- 2020-2021年新能源（双碳政策+电动车渗透率跃升+业绩兑现）
- 2023年AI（大模型突破+算力需求爆发+政策支持）
- 2025年DeepSeek概念（AI应用落地催化的结构性行情）

### 2.2 主线在时间轴上的演化阶段

```
萌芽期 → 扩散期 → 高潮期 → 衰退期
(少数机构) (机构跟进+游资介入) (散户涌入+媒体饱和覆盖) (资金撤离+逻辑弱化)
```

ETF轮动应该重点参与扩散期和高潮期早期。萌芽期确定性太低，衰退期风险收益比恶化。核心问题是**判断当前处于哪个阶段**——这需要定量数据。

## 三、定量识别：数据来源与指标体系

定量工作的目标：对全市场行业/主题板块持续输出一个「主线得分」，用于轮动排序。

### 3.1 定量数据的获取渠道

| 数据源 | 覆盖范围 | 成本 | 适用场景 |
|--------|---------|------|---------|
| **Tushare Pro** | A股日线、行业指数、资金流向、指数权重 | 免费(积分制，2000积分可覆盖日线+行业) | 行业/板块日频指标计算的主数据源 |
| **AKShare** | 行业板块、概念板块、资金流向、龙虎榜、涨停板数据 | 完全免费开源 | 概念板块数据（更贴近A股炒作现实）、涨停板统计 |
| **东方财富网页** | 板块资金流向、龙虎榜、涨停统计 | 免费（可爬取） | 补充验证、涨停家数快速查看 |

**关键取舍**：你不需要Level-2或Tick级数据来做主线识别。日频数据+行业/概念板块维度已经足够。原因：主线的时间尺度是月级别，日内的微观波动是噪音。

#### 3.1.2 特别注意：行业分类的选择

A股有两种行业分类体系，各有用途：
- **申万行业分类（31个一级行业）**：标准化、数据完整、适合回测和学术研究。Tushare有完整覆盖。
- **东方财富概念板块（300+个）**：更贴近A股实际的炒作聚合（如「机器人概念」「低空经济」），但板块边界模糊、成分股交叉严重。

建议**双层结构**：主策略用申万行业做回测和排序，概念板块用于验证和发现新方向（通过AKShare）。

### 3.2 定量指标体系

以下指标的目标不是各自打分然后加权，而是构成一条逻辑链——每个指标回答链条中的一个具体问题。

#### 3.2.1 第一步：方向确认——谁在涨？

**相对强度（Relative Strength）**

```
RS_i(t) = 行业i近N日累计收益 / 宽基指数近N日累计收益 - 1
```

- N取20日（月线级别）：确认中期趋势方向
- N取60日（季线级别）：确认是否进入主线候选

**核心判断**：连续N日RS > 0 + 趋势向上 → 方向确认。RS短暂为正但频繁正负切换 → 轮动而非主线。

**数据需求**：行业日线收盘价（Tushare `sw_daily` 或 `index_daily` 接口）。

#### 3.2.2 第二步：资金确认——钱真的在进吗？

这是一个容易被滥用和误读的领域。先厘清资金流向数据能告诉我们什么、不能告诉我们什么。

**资金流向的本质缺陷**：

A股所有免费的资金流向数据（东方财富、同花顺等）底层都是基于**逐笔成交的主动买卖分类**——把主动买入成交额记为流入、主动卖出成交额记为流出。这有几个问题：
1. **净流入/流出恒等于0**：每一笔成交有买方就有卖方，理论上的「资金净流入」在数学上不存在。实际计算的是「主动买-主动卖」，这反映的是交易发起方的积极性差异，而非资金的方向性移动。
2. **大单/小单分类是经验推测**：券商数据根据单笔成交金额反推是大户还是散户，不是真实的账户标识。准确性有限。
3. **行业板块维度更粗糙**：行业板块的资金流向通常是成分股汇总，噪声更大。

**结论：资金流向数据有价值，但必须理解其含义——它衡量的是「主动交易的方向偏斜」，而非真实的资金转移。** 把它作为确认信号而非判断依据。

**具体指标**：

| 指标 | 计算 | 含义 | 数据来源 |
|------|------|------|---------|
| 行业成交额占比 | 行业日成交额/全市场日成交额 | >15%才算有资金聚焦 | Tushare `moneyflow_hsgt` |
| 成交额占比趋势 | 20日均值是否持续攀升 | 资金在加速还是撤退 | 自行计算 |
| 行业相对换手率 | 行业换手率/全市场换手率 | 衡量交易活跃度的相对位置 | Tushare日线数据 |
| 大单净流入占比 | 行业大单净买入额/行业成交额 | 机构/大户偏向买入还是卖出 | AKShare `stock_sector_fund_flow` |

**大单净流入占比的阈值参考**（基于A股历史统计）：
- 持续 >5%：较强的机构买入意愿
- -5% ~ 5%：中性，没有明显的主动偏斜
- 持续 < -5%：机构在撤退

#### 3.2.3 第三步：广度确认——是全面开花还是个别权重股拉抬？

这是定量识别中最关键且最容易被忽略的一步。板块涨幅可以被少数大市值成分股绑架，形成虚假热度。

**指标一：板块内上涨股票占比（Advance Ratio）**

```
AR_i(t) = 行业i内当日上涨个股数 / 行业i内总个股数
```

- AR > 60% 且持续多日 → 板块内共识强，是真正的主线特征
- 板块指数涨但 AR < 50% → 少数权重股拉指数，虚假信号

**指标二：涨停家数**

A股涨停有特殊的信号含义——涨停代表了买方意愿被制度阻断，反映的是极端需求。

- 单板块连续3日涨停≥5家 → 该方向资金高度共识
- 涨停家数从高位持续下降 → 热度退潮

**数据来源**：AKShare 的 `stock_zt_pool_*` 系列接口，东方财富涨停板页面。

**指标三：成分股收益率离散度**

```
Disp_i(t) = 标准差(行业i内个股近N日收益率)
```

离散度的动态变化比绝对值更有信息量：
- 离散度收窄 + 板块上涨 → 资金在集中推升共识标的（扩散期特征）
- 离散度扩大 + 板块上涨 → 补涨股启动，全面扩散（高潮期特征）
- 离散度扩大 + 板块滞涨 → 龙头已熄火，补涨股勉强支撑，末端

**指标四：行业轮动速度（全局指标）**

```
Rotation_Speed(t) = 1 - Rank_Correlation(行业排名(t), 行业排名(t-N))
```

- 轮动速度低（排名稳定） → 市场有清晰主线
- 轮动速度高（排名剧烈变化） → 市场缺乏方向，追动量策略容易亏损

这个指标来自学术文献中对sector momentum的衡量方法。N取5日或10日。

#### 3.2.4 第四步：持续性确认——是趋势还是脉冲？

| 指标 | 含义 | 阈值 |
|------|------|------|
| 连续跑赢天数 | 连续RS>0的天数 | 连续10日+ → 趋势形成 |
| 区间最大回撤 | N日内从最高点的最大跌幅 | 控制在10%以内 → 趋势健康 |
| 创新高频率 | N日内创新高的天数占比 | >30% → 强势 |

#### 3.2.5 指标体系总结：逻辑链

```
RS确认方向 → 成交额占比确认资金聚焦
    → 上涨广度确认不是权重股拉抬
        → 离散度趋势确认扩散阶段
            → 持续性指标确认非脉冲
                → 主线得分 = f(上述指标)
```

不需要复杂的因子合成。一个简单的**条件筛选+排序**框架就可以：只有通过所有条件筛选的板块才进入轮动候选池，在候选池内按RS排序。这比机器学习黑箱更可解释，也比单一动量更稳健。

## 四、定性识别：信息源体系

定量告诉你是谁在涨，定性告诉你能不能持续涨。定性端的核心痛点不是「信息不够」，而是「信息太多」。散户最容易被海量垃圾资讯淹没——自媒体的标题党、涨停复盘、大V荐股——这些是噪音，不是信息。

### 4.1 定性信息的本质：三个层次

```
第一层：政策方向（为什么这个故事能被市场接受？）
    ↓
第二层：行业景气度（基本面能不能支撑价格？）
    ↓
第三层：市场叙事（市场在用什么语言讲述这个故事？）
```

三个层次必须保持一致，主线才成立。政策支持但行业景气度下行 → 炒概念，不可持续。行业景气度高但政策打压 → 估值压制，涨不动。政策和景气度都好但市场叙事没有形成 → 可能处于萌芽期，尚未被充分定价。

### 4.2 第一层：政策信息——最硬的信号来源

A股的政策敏感度远高于美股。市场主线的起点几乎总是政策信号。

#### 4.2.1 必看的一手政策来源（免费）

| 来源 | 内容 | 频率 | 怎么用 |
|------|------|------|--------|
| **国务院官网**（www.gov.cn） | 国务院常务会议纪要、国发文件 | 国常会每周一次 | 只看标题即可，扫到「大力发展」「重点支持」等关键词 |
| **发改委官网** | 产业政策、价格政策、规划 | 不定期 | 产业规划和价格改革类文件最关键 |
| **工信部官网** | 行业准入、技术路线、补贴 | 不定期 | 制造业/科技行业风向标 |
| **人民银行/证监会** | 货币政策、资本市场监管 | 不定期 | 影响整体风险偏好，定仓位用 |

**效率策略**：不需要每天刷。使用以下任一方式即可：
- 关注公众号「券商中国」「中国基金报」的早间资讯推送（每日7:00-8:00，约3分钟扫完标题）
- 使用萝卜投研或慧博的「政策资讯」聚合功能

#### 4.2.2 二手政策解读（过滤后使用）

- **券商宏观/策略报告的政策点评部分**：比自媒体靠谱，但倾向性不可忽略（券商基本不会唱反调）
- **财新网**：比大多数财经媒体更深入，但部分内容付费
- **华尔街见闻VIP**：实时政策解读，年费约500-1000元，性价比中等

**注意**：不需要深入解读每一个政策文件。政策信息的价值在于**识别方向性的变化**（「政策从打压转向支持」「补贴从退坡转向加码」），而非咬文嚼字。

### 4.3 第二层：行业景气度——基本面验证

确定了可能的政策方向之后，需要验证产业数据是否真的在改善。这是区分「主线」和「炒概念」的关键。

#### 4.3.1 行业高频数据来源

| 行业 | 核心跟踪指标 | 免费数据来源 |
|------|-------------|-------------|
| 新能源车 | 月度销量、渗透率 | 乘联会官网、中汽协 |
| 光伏 | 组件价格、新增装机 | 国家能源局、PVInsights |
| 半导体 | 全球销售额、设备出货 | SIA、SEMI（有延迟） |
| 消费电子 | 手机出货量 | 信通院 |
| 医药 | 医保谈判结果、集采品种 | 国家医保局 |
| 地产 | 30城销售面积、土拍 | 中指研究院（部分免费） |
| 基建 | 专项债发行进度、水泥出货率 | 财政部、数字水泥网 |
| 通用 | 工业企业利润（分行业） | 国家统计局 |

#### 4.3.2 效率策略：用券商研报「外包」景气度跟踪

你不需要亲自爬取所有行业数据。策略是：

1. **找到2-3家靠谱券商的行业首席**（不是策略分析师，是具体行业的研究员），定期看他们的周报/月报
2. 他们的工作就是持续跟踪行业高频数据，你只需要看结论和关键图表
3. 为什么只看2-3家？因为行业数据是公开的，不同分析师的判断差异主要在于解读角度，多看不增加信息量

**如何筛选靠谱分析师**：
- 看新财富/水晶球排名（有信息含量但不是充分条件）
- 更有效：看过去2年对行业拐点的判断是否准确（翻旧报告验证）
- 萝卜投研和慧博都支持按分析师筛选报告

### 4.4 第三层：市场叙事——这层最容易被量化交易者忽略

市场叙事不是「故事」或「炒作」，而是**市场参与者用来解释价格变动的共识性语言**。叙事的作用：
- 让机构有理由推荐（给基金经理的投委会一个说法）
- 让散户有理由参与（「AI革命」「国产替代」「碳中和」）
- 让资金形成合力（所有人用同一个故事做多）

叙事的可量化特征：
- 券商研报中某个关键词的出现频率是否在攀升
- 主流财经媒体是否在密集报道某个主题
- 雪球、微信公众号上关于该主题的讨论量

**关键判断**：叙事必须领先于价格或与价格同步。如果价格已经涨了50%才出现饱满叙事 → 大概率是接盘信号。

### 4.6 定性信息的防噪音策略

这是最重要的部分。大多数散户被信息淹死，不是被信息饿死。

**铁律**：
1. **永远先有定量信号，再查定性信息。** 不是读完研报再去看市场——而是看到某个板块在定量上出现异常信号（RS走强+成交额占比上升+广度扩散），然后去查「为什么」。这保证你的信息消费是需求驱动的，不会被推送信息牵着走。
2. **每天定性信息摄入总量控制在20分钟以内。** 超过这个时间，边际信息质量急剧下降。
3. **不刷财经APP的信息流。** 信息流的算法优化目标是你的停留时间，而非你的投资回报。它们会给你推送最刺激但不一定最有用的内容。
4. **以研报替代资讯。** 一篇深度行业报告的信息密度远高于100条财经快讯。5篇深度报告/周 >> 500条新闻/周。

## 五、定性与定量的集成：实战工作流

### 5.1 周度循环（建议每周日执行）

```
Step 1（定量，10分钟）：运行脚本，输出本周主线候选池
    - 输出项：RS_20d > 0 的行业/概念板块
    - 按成交额占比排序，取前5

Step 2（定量，自动）：验证候选池的广度、持续性
    - 过滤掉上涨广度<50%或连续跑赢<10天的

Step 3（定性，15分钟）：对通过过滤的板块，查「为什么」
    - 萝卜投研搜索「[板块名称] 行业 研报 2026」
    - 看最近2篇深度报告的结论和景气度判断
    - 快速确认政策方向是否友好

Step 4（决策）：输出本周轮动配置
    - 主线板块 → 高权重配置
    - 候选板块 → 低权重试探
    - 无主线 → 防御配置（债券ETF或货币）
```

### 5.2 日度检验（每天收盘后10分钟）

- 运行脚本，检查持仓板块的上涨广度、量价是否保持健康
- 如有异常（AR骤降+RS转负），进入降仓判断
- **不需要每天做定性工作**——周度把握方向，日度看定量信号即可

### 5.3 数据流架构

```
Tushare/AKShare（定量数据）──→ 本地数据库 ──→ 指标计算脚本 ──→ 主线得分
                                                              │
萝卜投研/政策源（定性信息）──→ 笔记/Obsidian ────────────────→ 置信度调整
```

定性信息不需要结构化存储进数据库。用Obsidian做定性笔记，每周末做一次逻辑整理即可。

## 六、萝卜投研的定位与局限性

### 6.1 萝卜投研擅长什么

- **研报搜索和过滤**：按行业、关键词、时间搜索研报，速度快
- **行业/公司数据看板**：股价、估值、财务数据的可视化
- **产业链图谱**：展示上下游关系，帮助理解一个主题涉及哪些标的
- **AI摘要**：自动提取研报中的关键结论（节省大量阅读时间）

### 6.2 萝卜投研的问题

- **研报覆盖率不如慧博**：萝卜投研偏向头部券商，二三线券商和小券商的研报覆盖不全。但这对主线识别影响不大——你看的就是主流叙事，头部券商的覆盖已足够。
- **没有任何定制化/API输出**：数据只能在App/网页上查看，不能导入你的量化框架
- **价格**：个人版几百元/年，远低于万得/Choice

### 6.3 如果萝卜投研不够用，补什么

**不要直接升级到万得**。补的顺序：
1. **慧博免费版/低价版**（约1000-2000元/年）：研报覆盖率更高，适合在你需要深挖某个具体行业时使用
2. **华尔街见闻VIP**（约500元/年）：实时政策解读+市场叙事跟踪
3. **Choice终端**（约3000元/年）：只有在你确定需要更强的数据导出/API能力时才考虑

对于「识别主线」这个任务，萝卜投研的模式——研报聚合+AI摘要——是正确的。问题只是覆盖够不够深。

## 七、与ETF轮动策略的关系

主线识别是ETF轮动的前置模块。两者的关系：

```
主线识别 ──→ 候选池（哪些板块值得轮动）
    │
    ├── 主线板块 → 高权重（动量因子 + 基本面置信）
    ├── 候选板块 → 低权重试探
    └── 无主线   → 防御配置
         │
         ▼
    ETF轮动策略（nanoquant）—— 在候选池内执行具体排序和调仓
```

已有的ETF动量质量因子（[[ETF动量质量因子]]）和筹码结构因子（[[筹码结构与因子构造]]）是轮动引擎的核心组件。主线识别提供的是一个**筛选层**——决定哪些板块进入引擎、以什么权重进入。

**关键设计原则**：主线识别和轮动引擎要解耦。不要因为某个板块定性信息很强就强行持有（「逻辑正确」可能是你的确认偏误），也不要因为定性信息缺失就完全忽略（定量信号可能领先于定性认知）。

解耦方式：定性信息调节的是**置信度**和**仓位上限**，而非是否持有的二元决定。

## 八、总结

1. 主线识别 = 定性（逻辑方向）+ 定量（市场确认），两条腿缺一不可
2. 定量端免费：Tushare + AKShare 已足够覆盖行业/概念板块的全部指标需求
3. 定性端的核心不是花钱买更多信息，而是**用定量信号驱动定向搜索，避免被推送信息淹没**
4. 萝卜投研对主线识别够用，如果不够，先补慧博而不是万得
5. 最终目标不是「判断对的板块」，而是构建一个**可复制的周度评估流程**，嵌入ETF轮动的上游
