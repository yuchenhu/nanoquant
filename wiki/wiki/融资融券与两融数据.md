# 融资融券与两融数据

> 创建日期：2026-07-06

---

## 一、概述

融资融券是A股唯一合法的杠杆交易渠道，连接着信用机制、保证金制度、市场情绪与量化数据。在知识网络中，两融数据是[[筹码结构与因子构造]]中"筹码分布"的杠杆维度延伸，也是[[2015年股灾始末]]和[[2024年微盘股流动性危机]]中强平螺旋机制的底层制度基础。ETF两融数据还可嵌入[[ETF动量质量因子]]中作为"杠杆资金流向"维度。

---

## 二、核心机制：双层结构与资金流

### 2.1 融资（Margin Buying）

**本质**：以信用账户中的资产为抵押，向券商借钱买入证券。

```
保证金存入 → 券商开设信用账户
    ↓
融资买入指令 → 券商出借资金（自有资金或银行拆借）
    ↓
买入的证券 + 原有资产 = 担保品（存放于信用账户，券商有处置权）
```

### 2.2 融券（Short Selling）

**本质**：以信用账户资产为抵押，向券商借入证券卖出，预期跌后买回归还。

```
保证金存入 → 借入证券 → 二级市场卖出获得现金（冻结在信用账户）
    ↓ 等待下跌
买入同数量证券归还 → 差价 = 盈利（扣除融券利息）
```

**A股融券的特殊约束**：
- **券源端**：仅来自券商自营盘持仓 + 证金公司转融通（其他机构出借），供给严重受限
- **卖出端**：融券卖出价不得低于最新成交价（提价规则，防止加速砸盘）
- **监管端**：2023年后进一步收紧，限制战略投资者出借
- **结果**：全市场融券余额约226亿（2026-07-03），占两融总额仅0.75%，近乎可忽略的体量

### 2.3 资金性质分类：存量 vs 流量

融资余额（存量）与融资买卖（流量）是不同维度的信息：

```
融资余额 = 3万亿   ← 存量：市场里同时持有的融资仓位总量
融资买入 ≈ 1500亿/日 ← 流量：每天新进的融资资金
融资偿还 ≈ 1500亿/日 ← 流量：每天离场的融资资金
净变化   ≈ ±30亿/日  ← 进出抵消后的余额变化
```

余额大 + 变化小 + 流入流出都很大，三者同时成立且互不矛盾：
- 余额大 = 同时在场的人多
- 变化小 = 每天进的人和出的人数量大致相等（独立个体的换手，不是同一笔钱的进出）
- 判断资金短线与否的关键指标是 **融资买入/融资余额比值**：

| 买入/余额比 | 资金性质 | 典型标的 |
|:---------:|---------|---------|
| >25% | 游走型：超短线杠杆，高频换手 | 深科技（31%）、TCL科技 |
| 5-25% | 混合型：中等活跃度 | 中信证券 |
| <5% | 沉淀型：长期杠杆持仓，低周转 | 贵州茅台（2.2%）、中国平安（1.3%） |

### 2.4 沉淀型融资的经济逻辑——为何付费借钱持股

根本原因：**银行个人贷款禁止流入股市**，券商融资是唯一合法渠道。

沉淀型投资者的算盘：
```
假设标的预期年化回报 12%，券商融资利率 6%：

不加杠杆：100万 × 12% = 12万收益 → ROE 12%
1:1杠杆：200万 × 12% - 100万 × 6%利息 = 18万收益 → ROE 18%

只要持有期预期回报 > 融资利率，加杠杆就是正EV。
```

除了利差套利，还有**时间维度**的理由：错误定价的窗口不等人。融资买的是"不用等攒钱就能建立目标仓位"的时间权——利息是入场门票。

## 三、保证金制度与强平机制

### 3.1 保证金比例的历史演变

| 时期 | 最低保证金比例 | 最大杠杆倍数（总仓位/自有） |
|------|:------------:|:----------------------:|
| 2015股灾前 | 50% | 3.0x |
| 2015年11月（收紧） | 100% | 2.0x |
| 2023年8月至今 | **80%** | **2.25x** |

**注意**：80%是最低线，券商可对高波动个股设置更高比例。2023年下调后杠杆仍远低于2015年水平，这是当前两融系统脆弱性低于2015年的核心原因。

### 3.2 维持担保比例——平仓生命线

```
维持担保比例 = (现金 + 证券市值) / (融资负债 + 融券负债)

关键阈值：
  300%  ← 提取线：超出部分可转出
  200%  ← 初始状态（1:1杠杆时）
  150%  ← 警戒线：不能再融资买入
  130%  ← 平仓线：T+1补足保证金，否则 T+2强平
  110%  ← 即时平仓线：无需等待
```

**实例验算**（融资保证金比例100%，即1:1杠杆）：
```
存入100万，融资买入100万 → 总资产200万，负债100万
  初始比例 = 200/100 = 200%

股价跌20% → 总资产180万 → 比例 = 180%（安全）
股价跌35% → 总资产130万 → 比例 = 130%（触发预警！）
股价跌40% → 总资产120万 → 比例 = 120%（开始强平）
股价跌50% → 总资产100万 → 比例 = 100%（本金清零）
```

### 3.3 强平螺旋——正反馈崩溃机制

```
股价下跌 → 维持担保比例下降 → 触及130%平仓线
    ↓
券商强平卖出持仓 → 额外卖压砸向市场
    ↓
股价进一步下跌 → 更多账户触及平仓线 → 更多强平
    ↓
正反馈螺旋：卖压 → 下跌 → 强平 → 更多卖压 → 更深下跌
```

这与普通下跌的本质区别：**强平产生的卖盘不是基于"看空"判断，而是无选择权的被迫卖出**——价格不重要，必须卖。这是流动性危机的核心机制，直接连接到[[2015年股灾始末]]和[[2024年微盘股流动性危机]]。

### 4.1 可用接口

| 接口 | 内容 | 粒度 | 字段 |
|------|------|------|------|
| `margin` | 市场级汇总 | 日频，按交易所（沪深京） | rzye, rzmre, rzche, rqye, rqmcl, rzrqye, rqyl |
| `margin_detail` | 个股/ETF明细 | 日频，全量标的（4000+条/日） | rzye, rqye, rzmre, rzche, rqchl, rqmcl, rzrqye |
| `margin_secs` | 两融标的列表 | 盘前更新 | ts_code, name, exchange |
| `margin_target` | 历史标的变动 | 按日期+标的 | ts_code, mg_type, is_new, in_date, out_date |

### 4.2 字段速查

| 字段 | 含义 | 单位 |
|------|------|------|
| rzye | 融资余额（月末/日末存量） | 元 |
| rzmre | 融资买入额（当日发生） | 元 |
| rzche | 融资偿还额（当日发生） | 元 |
| rqye | 融券余额 | 元 |
| rqmcl | 融券卖出量（当日） | 股 |
| rqyl | 融券余量（存量） | 股 |

### 4.3 数据量级

- `margin`：每天3条，极轻量
- `margin_detail`：每天4000+条，按月拉取约12万条，完全在个人量化可处理范围内
- T+1盘后更新，时效性足够日频策略
- 免费（Tushare积分够用时）

## 五、量化因子构造

### 5.1 因子速查表

| 因子/信号 | 构造方法 | 阈值/注意事项 |
|-----------|---------|-------------|
| 融资余额变化率 (N日) | `(rzye_t - rzye_{t-N}) / rzye_{t-N}` | 用5日或20日均线平滑，不用单日 |
| 融资买入额/成交额比 | `rzmre / 当日总成交额` | >15% 过热，<5% 冰点；极端值比均值有用 |
| 融资拥挤度 | `rzye / 流通市值` | >15% 高拥挤预警，>20% 极度拥挤雷区 |
| 融资买卖比 | `rzmre / rzche` | >1 净买入，<1 净偿还；均线平滑后用 |
| 融资-价格背离 | `sign(Δrzye) ≠ sign(Δprice)` | 价涨融资降=动能衰减；价跌融资增=潜在支撑 |
| 融券异常突增 | `rqye_t / avg(rqye_{t-20}) > 2` | 需排除ETF做市商对冲行为 |
| 两融交易参与度 | `(rzmre + rzche) / 总成交额` | 衡量杠杆资金活跃度 |

### 5.2 融资信号的时间变异性（Regime-Dependent Quality）

融资余额变化由两个通道驱动，其信号质量随市场状态变化：

**主动通道**：投机者看涨 → 主动融资买入 → 推动价格（同步/领先信号）
**被动通道**：股价涨 → 抵押品升值 → 维持担保比例上升 → 释放可融资额度 → 已有持仓者被动加仓（滞后/衰减信号）

信号解读需结合市场位置：
- **上涨初期/中期**：主动通道主导，融资增加 = 有效看多信号
- **上涨末期/加速段**：被动通道叠加，融资边际质量和预测力衰减
- **高拥挤环境下**（融资/流通市值 >15%）：继续增加的融资余额边际预测力急剧下降——新进资金多为被动释放催生的跟随者
- **下跌中**：被动通道反向——融资余额下降可能是强平造成的被动减仓，而非主动看空

因此融资信号不能一刀切地判定"领先"或"滞后"，需要结合价格位置、拥挤度、上涨持续时间做**状态依赖加权**。

### 5.3 对 ETF 轮动策略的应用

1. **行业聚合融资信号**：将 margin_detail 按行业聚合，计算行业融资余额周度变化率 + 融资买入额占比，作为 ETF 动量质量因子的"杠杆资金流向"子维度
2. **拥挤度预警**：计算 ETF 持仓成分股的融资拥挤度加权平均，大面积 >15% → 踩踏风险上升 → 降低仓位或回避
3. **情绪温度计**：全市场融资买入/成交比作为辅助情绪指标，配合[[识别市场主线]]框架——主线行业应同时满足"基本面逻辑 + 融资资金持续流入"
4. **融资均线金叉/死叉**：5日融资余额均线 vs 20日均线，金叉确认趋势，死叉资金撤退。但顶部时此信号失效（2015年教训）
