# 基金净值与申赎机制

> 关联页面：[[机构抱团与踩踏]] [[ETF申赎套利与AP机制]] [[打板策略]]

---

## 一、核心等式

```
基金总资产 = ∑(每只持仓股 × 最新市价) + 现金 + 应收款项 - 应付款项

单位净值(NAV) = 基金总资产 / 基金总份额

基金总规模 = 基金总份额 × 单位净值
```

所有衍生问题从这三个等式展开。

---

## 二、申购：资金进来，净值不变，份额增加

示例：基金当前总资产10亿，总份额5亿份，单位净值2.00元。外部投资者申购1亿元。

- 申购价格 = T日收盘后净值 = 2.00元
- 新增份额 = 1亿 / 2.00 = 0.5亿份

| 变量 | 申购前 | 申购后 |
|------|--------|--------|
| 总资产 | 10亿 | 11亿（现金+1亿，持仓不变） |
| 总份额 | 5亿份 | 5.5亿份 |
| 单位净值 | 2.00元 | 2.00元（不变） |

**结论：申购不改变净值。** 分子和分母同步增加。

---

## 三、赎回：资金离开，净值不变，份额减少

同一基金，有人赎回5000万元。

- 赎回份额 = 5000万 / 2.00 = 2500万份

| 变量 | 赎回前 | 赎回后 |
|------|--------|--------|
| 总资产 | 11亿 | 10.5亿（现金-5000万） |
| 总份额 | 5.5亿份 | 5.25亿份 |
| 单位净值 | 2.00元 | 2.00元（不变） |

**结论：赎回不改变净值。** 分子和分母同步减少。

---

## 四、净值变动的唯一来源：持仓资产价格变动

| 事件 | 对净值的影响 | 说明 |
|------|-------------|------|
| 持仓股价格上涨 | 净值↑ | 总资产↑，份额不变 |
| 持仓股价格下跌 | 净值↓ | 总资产↓，份额不变 |
| 投资者申购 | 不变 | 资产和份额同增 |
| 投资者赎回 | 不变 | 资产和份额同减 |
| 收取管理费/托管费 | 净值↓ | 每日从资产中计提（微量持续） |
| 基金现金分红 | 净值↓ | 现金流出，份额不变（跳跃式下跌） |
| 持仓股分红（除权） | 不变 | 股价↓+现金↑，总资产不变 |

## 五、申购如何间接推高净值（抱团正反馈机制）

虽然申购本身不改变净值，但存在间接传导：

```
申购 → 现金占比上升 → 基金经理被迫建仓（持现=跑输基准）
     → 买入持仓股 → 推高股价 → 持仓市值↑ → 净值↑
     → 排名提升 → 更多申购 → 回到起点
```

中间环节有1-3个交易日的滞后（基金经理拿到钱到完成建仓）。

## 六、ETF vs 普通基金的申赎差异

| 维度 | 普通开放式基金 | ETF |
|------|--------------|-----|
| 申购方式 | 投资者给现金 | AP给一篮子股票（按PCF清单） |
| 基金经理行为 | 需要拿现金去买股票（1-3天） | 不需要建仓——AP已经把股票给了 |
| 最小申赎单位 | 通常1元起 | 数十万到数百万份（篮子） |
| 申赎效率 | 受建仓滞后影响 | 实物交换，无冲击成本 |
| 对持仓的影响 | 新资金可能影响持仓结构 | 申赎不改变已有持仓结构 |

ETF基金经理不需要因为资金流被迫在市场交易——AP的实物申赎机制将交易执行外包给了AP，基金经理只负责按PCF清单管理持仓权重。

详见 [[ETF申赎套利与AP机制]]。

## 七、净值计算中的隐藏成本

| 费用项 | 费率 | 收取方式 |
|--------|------|---------|
| 申购费 | 1%-1.5%（前端） | 申购时扣除（申购1万≈9850-9900元进入基金） |
| 赎回费 | <7天1.5%，<30天0.5%-0.75% | 赎回时扣除 |
| 管理费 | 1%-1.5%/年 | 每日从净值中计提（曲线上不显「某天暴跌」，持续微量） |
| 托管费 | 0.2%-0.25%/年 | 同每日计提 |

**量化打板中「变相T+0」的前提**：持有底仓超过7天/30天以上，赎回费已降至极低。否则打板的微薄α瞬间被赎回费吞噬。

---

## 八、累计净值 vs 单位净值

```
累计净值 = 单位净值 + 历史每份累计分红
```

分红时基金卖出部分持仓→获得现金→分给持有人→单位净值下跌（总资产减少），但持有人的总价值不变（拿回了现金）。用累计净值而非单位净值判断历史表现，才能避开分红的干扰。
