# ETF策略风险指标与做空工具

> 创建：2026-07-14

---

## 第一部分：预测性风险因子——动量信号的断路器

> **关键区分**：你要的不是事后评价指标（MDD、Sharpe、VaR是回测后看的），而是**预测性因子**——在回撤发生**之前**给出信号，触发后直接覆盖动量信号，平仓/减仓。

### 一、架构逻辑：两信号串联

```
日频运行：
  ETF动量打分（alpha信号）──→ 风控开关（风险因子）──→ 最终仓位
                                    │
                    如果任一风险因子触发 → 仓位归零，不管动量多强
```

这不是「动量打分里加一个风险惩罚项」——是两层独立决策。alpha管选什么、配多少；风控开关管「现在能不能持仓」。两层的优化目标完全不同。

### 二、第一性原理：风险因子为什么能预测回落？

回落不是随机事件。回落前几乎必然出现以下至少一个前兆：

| 前兆 | 因果链 | 领先时间 |
|------|--------|---------|
| 波动率从低水平跳升 | 市场共识瓦解 → 参与者分歧扩大 → 价格剧烈摆动 | 1-5天领先 |
| 相关性突变上升 | 正常时期分散的风险因子同时激活 → diversification fails | 0-3天领先（同步性最强） |
| 流动性恶化 | 做市商退缩价差 → 小额订单即可推动价格 → 波动被放大 | 0-3天领先 |
| 资金流向逆转 | 之前推动上涨的资金停止流入/开始流出 → 买方力量衰竭 | 3-10天领先（最好用） |
| 信用/利率市场先行 | 债市参与者通常比股市更专业、更早反应宏观风险 | 5-20天领先 |

**核心洞察**：你要找的不是回落本身，而是「市场脆弱性上升」——当脆弱性很高时，任何小冲击都可能引发大回落。所以风险因子实际上是在度量「脆弱性水平」，不是在预测「冲击何时发生」。

### 三、量化表达层：可用的预测性风险因子

以下因子按「领先性 × 信噪比」排序，每个给出具体公式、A股数据来源、阈值建议。

---

#### 因子1：已实现波动率分位数（最基础，必须有）

```
RV_20d_rank = rank(RV_20d_today, among past 252 values) / 252

其中 RV_20d = sqrt(252 × mean(r²_1d))，20日滚动年化波动率
```

这是风险因子的骨架。波动率聚集（volatility clustering）是金融市场最稳健的统计规律——高波动之后还是高波动，且高波动时期回撤频率和幅度显著放大。

```
阈值：
  RV_20d_rank > 0.80 → 高风险regime，仓位上限 50%
  RV_20d_rank > 0.90 → 仓位上限 25%
  RV_20d_rank > 0.95 → 仓位归零
```

但你不可能只靠这一个。因为它有两个致命弱点：
1. 滞后：波动率上升是结果，你需要更早的信号
2. 跟动量的信息重叠：动量策略在高波动期可能已经在减仓了，你再按vol减仓是重复计数

所以需要下面的补充因子。

---

#### 因子2：波动率的波动率（Vol of Vol）

```
vol_of_vol_5d = std(RV_5d_over_past_60d)

其中 RV_5d 用5日窗口（比正常vol窗口更短）
vol_of_vol_5d 度量vol自身的抖动程度
```

正常上涨时vol平稳或缓慢上升。**回落前vol经常出现异常跳变**——不是vol高，而是vol从低突然跳到高。这是市场从「共识」切换到「分歧」的痕迹。

```
阈值：
  vol_of_vol_5d > its_90th_percentile (过去252日) → 预警
  vol_of_vol_5d > its_95th_percentile → 减仓50%
```

A股数据：直接用ETF本身的日收益率算，不需要额外数据源，Tushare `fund_daily` 够用。

---

#### 因子3：横截面离散度（Cross-Sectional Dispersion）

```
CS_dispersion = std(r_i)  across all ETFs in universe on day t
              即当日所有候选ETF收益率的标准差

rolling CS_dispersion_20d = mean(CS_dispersion, past 20 days)
```

**因果链**：当所有ETF同涨同跌时（dispersion低），市场有清晰的驱动力（某个因子/宏观变量主导）。当dispersion突然跳升时，意味着驱动力在瓦解——有些板块被抛弃，有些还在涨——这是趋势终结的前兆。

**注意**：这个因子对ETF轮动策略特别重要，因为它直接描述你选池的「内部一致性」。

```
阈值（相对自身历史）：
  CS_dispersion_today > CS_dispersion_90th_pct (252日) → 预警
  连续3日 > 80th_pct → 减仓50%
```

Tushare：自己算。拿你ETF候选池的所有日收益率求std，简单到不需要外部数据。

---

#### 因子4：市场广度（Market Breadth）

不是看大盘指数涨跌，而是看「有多少成分股在涨」。

```
# 用ETF跟踪的基准指数成分股（如沪深300成分股）
breadth_ma_long  = 百分比_of_成分股_价格高于_120日均线
breadth_ma_short = 百分比_of_成分股_价格高于_20日均线

breadth_divergence = breadth_ma_short - breadth_ma_long
```

**因果链**：牛市末期常见指数还在涨，但越来越多的成分股开始跌——「指数被大市值拖着走，小票先塌了」。牛市健康的标志是broad participation，广度恶化是回调前最可靠的领先指标之一。

```
阈值：
  breadth_ma_short < 40% 且 指数仍在MA20上方 → 背离预警 → 减仓50%
  breadth_ma_short < 30% → 无论指数怎么样 → 清仓
  breadth_divergence < -20%（即短期广度远低于长期广度）→ 结构性恶化 → 清仓
```

Tushare：需要成分股日线数据（`index_member_all` + `daily`）。对于ETF轮动，可以简化为：用候选池内所有ETF相对各自MA20的百分比，代替个股广度。

#### 因子5：资金流向（最希望有，但数据获取难）

```
flow_signal = (OBV_5d_change / average_dollar_volume_20d)

简化版：
moneyflow_ratio = sum(volume × close的位置分数, past 5 days)
其中 close的位置分数 = (close - low) / (high - low)，表示资金是在高位还是低位成交
```

**因果链**：价格可以被操纵，成交量可以被对倒，但**成交加权价格位置（VWAP偏离）** 很难伪装。如果过去5日均在低位成交（卖方主导），而价格还没跌——这是买家正在撤退的信号。

Tushare：`moneyflow` 接口可以直接拿资金流向，但数据质量和覆盖范围需要验证。不需要完美数据——你看的是方向性变化，不是绝对水平。

#### 因子6：VIX / 隐含波动率（美股可选，A股替代）

**VIX的核心价值不是「VIX多高」，而是「VIX相对背景水平有没有异常上升」。**

```
VIX_signal:
  VIX_today > VIX_ma_10d × 1.2（即10日均线上穿20%）→ 预警
  VIX_today > 25 且 VIX_today > VIX_ma_10d × 1.3 → 减仓50%
  VIX futures curve 从contango转为backwardation → 清仓
```

A股替代方案（没有iVIX数据时）：

```
替代1：用50ETF的30日已实现波动率分位数（就是因子1）
替代2：用股指期货隐含波动率——从当月连续合约的日振幅反推
  iv_proxy = (daily_range / close) × sqrt(252) / sqrt(2/π)
  （不精确但方向性正确）
替代3：国债期货波动率（T/ TF）上跳 → 债券市场波动先行 → 股市风险即将上升
```

**VIX v2.0 —— 更重要的洞察**（2026-07-14）：VIX的期限结构（contango/backwardation）本身**也是预测因子**。这一点第一次写wiki时已经覆盖，在3.2特征2中详细展开。

#### 因子7：信用利差（A股替代：产业债信用利差）

```
credit_spread = 中债AA级产业债收益率 - 同期限国债收益率

credit_spread_change_5d = credit_spread_today - credit_spread_5d_ago
```

**因果链**：债券投资者比股票投资者更早、更准确地对宏观风险定价。信用利差突然走阔意味着债市认为违约风险上升——这会传导到股市（企业融资成本上升→盈利预期下调→股价下跌）。

```
阈值：
  credit_spread_change_5d > 20bps → 预警
  credit_spread_change_20d > 50bps → 减仓50%
  credit_spread > 90th_pct (5年历史) → 长期恶化，减仓70%
```

Tushare：没有直接的信用利差数据。替代：Wind/Choice有；或者用 Shibor/国债利差做近似（但信息量弱很多）。

#### 因子8：北向资金（A股特有，极重要）

```
north_flow_5d = sum(北向净买入, past 5 days)
north_flow_ratio = north_flow_5d / avg_daily_turnover_20d

north_flow_reversal = north_flow_5d < 0 且 north_flow_20d > 0
                     （20日累计仍为正但5日开始净流出 → 转向信号）
```

北向资金是A股最重要的边际定价资金之一。历史上多次验证：北向持续净流出领先大盘回落5-10个交易日。

```
阈值：
  north_flow_5d < 0 且 |north_flow_5d| > avg_abs_daily_flow × 2 → 预警
  north_flow_10d < 0（连续10日净流出）→ 减仓50%
  north_flow_reversal 且 CS_dispersion > 80th_pct（叠加信号）→ 清仓
```

Tushare：`moneyflow_hsgt` 接口有沪深港通资金流向数据。

---

#### 因子9：两融余额变化（A股情绪温度计）

```
margin_change_5d = (margin_balance_today - margin_balance_5d_ago) / margin_balance_20d_avg

# 更关键的是融资买入额占比
margin_pct = 融资买入额 / 总成交额（rolling 5 day average）
```

**因果链**：
- 融资余额增加 = 杠杆资金看多 = 正常
- 融资余额在指数横盘/微跌时仍在增加 = 散户扛单 = 危险（强平螺旋的前兆）
- 融资买入占比异常高（>12%）+ 指数下跌 = 杠杆资金被套 → 随后可能强平加速下跌

```
阈值：
  margin_pct > 12% 且 index_return_5d < 0 → 减仓50%
  margin_change_5d < -5%（融资余额快速下降）→ 强平进行中 → 立刻清仓
```

Tushare：`margin` 和 `margin_detail` 接口。

---

#### 因子10：涨跌停情绪（A股特有，简单但有效）

```
limit_down_count = 全A当日跌停家数
limit_down_pct = limit_down_count / total_listed_stocks

limit_up_down_ratio = 涨停家数 / 跌停家数（rolling 5 day）
```

A股涨跌停是恐慌的即时温度计。跌停家数从个位数跳升到几十家→恐慌在扩散。

```
阈值：
  limit_down_count > 50（全A）→ 预警
  limit_down_count > 100 → 减仓50%
  limit_down_count > 200 且 连续2日 → 清仓
  limit_up_down_ratio < 0.5（跌停多于涨停一半）连续3日 → 清仓
```

Tushare：`limit_list` 或 `limit_list_d`。

### 四、因子组合架构：不是所有因子都跑

上面列了10个因子，但你不能全用。选因子有两条铁律：

1. **信息源不重叠**：两个因子如果高度相关，选信噪比高的那一个，扔掉另一个
2. **A股实际可获取**：没有数据源的因子 = 不存在

**推荐组合（A股ETF轮动，数据都可获取）**：

```
核心三层：

Layer 1（必有，最敏感）：
  • RV_20d_pct（波动率分位数）
  • vol_of_vol_5d（波动率跳变）

Layer 2（广度/结构，信息独立于Layer 1）：
  • breadth_ma_short（候选ETF中在MA20上方的比例）
  • CS_dispersion（ETF横截面离散度）

Layer 3（资金/情绪，领先性最强）：
  • north_flow_5d（北向5日累计）
  • margin_pct（融资买入占比）
```

**表决规则**（硬编码，不调参）：

```
任何1个Layer 3因子触发 → 仓位上限 50%（no matter what momentum says）
任何2个因子触发（跨Layer）→ 仓位上限 25%
任何3个因子触发 → 仓位归零
Layer 1两个同时高阈值触发 → 仓位归零（vol + vol_of_vol同时爆）
```

### 五、评价标准：风险因子的物理评价体系

#### 5.1 关键认知：风险因子 ≠ 排序因子，评价方法必须不同

动量因子是**排序因子**——你把候选ETF排序，long top、short bottom，spread的收益率直接告诉你因子好坏。评价逻辑是一条轴：排序→分组→算spread。

风险因子是**二元开关**——触发与否，仓位砍/留。评价逻辑完全不同：开关拨到ON之后，市场实际发生了什么？

这引出一个关键设计约束：**风险因子不应该预测正常时期的收益，只应该预测尾部风险。** 好的风险因子在正常市场应该保持沉默（OFF），在危险来临前亮灯（ON）。如果你拿factor value对forward return跑线性回归找IC，会得出风险因子IC≈0的结论——但这恰恰说明它好，因为它在正态区间不工作、只在尾部激活。

所以不能用 IC/rankIC 那套来评价风险因子。需要一套物理意义完全不同的指标体系。

#### 5.2 风险因子的"spread"：条件收益差（RAS）

动量因子 → top3-bottom3 spread，单位是%收益率。

风险因子 → 把全历史按因子状态切成两组：

```
Safe组（未触发）：仓位正常持有，获取动量策略原始收益
Danger组（已触发）：空仓/减仓，收益为0（或减仓后的持有部分收益）

RAS = E[r_fwd_Nd | Safe] - E[r_fwd_Nd | Danger]
```

单位是%，物理意义：**这个因子帮你在未来N天内，平均避开了多少%的下跌（或少赚了多少%的上涨）。**

但RAS一个数字不够。它在牛市里可能是负的（减仓错过了上涨），在熊市里应该是正的（减仓躲过了下跌）。所以必须拆开看：

```
RAS_total   = E[r_fwd_Nd | Safe] - E[r_fwd_Nd | Danger]
RAS_bull    = E[r_fwd_Nd | Safe, mkt_up] - E[r_fwd_Nd | Danger, mkt_up]
RAS_bear    = E[r_fwd_Nd | Safe, mkt_down] - E[r_fwd_Nd | Danger, mkt_down]
             （mkt_up/down 用同期宽基指数涨跌二分）
```

**物理意义的判读：**

```
理想风险因子：
  RAS_total  ≈ 0         （不拖累牛市收益）
  RAS_bull   ≈ 0 或略负  （牛市时触发少，错过有限）
  RAS_bear   ≫ 0         （熊市时大量避险）

垃圾风险因子：
  RAS_total  < 0         （总体在帮倒忙）
  RAS_bull   ≪ 0         （牛市频繁误触发，大量踏空）
  RAS_bear   ≈ 0         （熊市来了反而没触发）

无信息风险因子：
  RAS_total  ≈ 0, RAS_bull ≈ 0, RAS_bear ≈ 0
  （跟掷硬币没区别，开关是随机的）
```

#### 5.3 四个物理指标，替代模糊的「信噪比」

**指标1：避险收益（RAS_bear）**

```
RAS_bear = 熊市环境下，Safe组平均收益 - Danger组平均收益
         （N=10或20天）

门槛：RAS_bear > 2%（20日口径）→ 有实际避险价值
      RAS_bear > 5% → 优秀
```

物理意义：市场下跌的20天里，这个因子帮你少亏了5%以上。

**指标2：踏空代价（Profit Sacrifice = RAS_bull 的绝对值）**

```
Profit_Sacrifice = |RAS_bull| = |E[r_fwd | Safe, bull] - E[r_fwd | Danger, bull]|

门槛：Profit_Sacrifice < 1%（20日口径）→ 可接受
      Profit_Sacrifice > 3% → 牛市里频繁误触发，因子在吃掉你的alpha
```

物理意义：牛市里你因为被这个因子赶出场，平均少赚了多少。这就是你买保险的"保费"。

**指标3：危机捕获率（Crisis Capture Rate, CCR）**

```
定义：历史回撤>5%的事件中，因子在回撤开始前N天内触发的比例

CCR = count(因子在回撤前触发且方向正确) / count(总回撤事件)

门槛：CCR > 50% → 有效（一半以上的大回撤有预警）
      CCR > 70% → 优秀
```

这是最直观的物理指标：**历史上每次摔跤之前，闹钟响了几次？**

证伪条件：如果某个因子CCR=0%（历史上从未预警任何一次大回撤），它不是风险因子，它是噪音。

**指标4：假警报率（False Alarm Rate, FAR）**

```
FAR = count(触发后N日内未出现>2%回撤) / count(总触发次数)

门槛：FAR < 40% → 可接受
      FAR > 60% → 狼来了太多次，你会无视它
```

物理意义：这个闹钟响了10次，有几次是误报？误报率太高=你最终会关掉闹钟=因子失效。

> **CCR和FAR是跷跷板**：阈值设得越敏感→CCR越高（抓住更多危机）→但FAR也越高（误报更多）。调优的目标不是两个都完美，是在你心理可承受的FAR下最大化CCR。业内实操：FAR 30-40%、CCR 60-70%是一个务实的目标区间。

#### 5.4 终极评价：净收益贡献

以上四个指标最终收敛到一个数：

```
Net_Benefit = (RAS_bear × 熊市占比) - (Profit_Sacrifice × 牛市占比) - (FAR × 踏空概率 × 平均踏空幅度)

简化版（年度口径）：
Net_Benefit_annual = 因子触发帮你避开的年化损失 - 因子触发让你错过的年化收益

如果 Net_Benefit_annual < 0 → 扔掉这个因子，它在帮倒忙
如果 Net_Benefit_annual ∈ [0, 2%] → 边际有用，但可能不值得复杂度
如果 Net_Benefit_annual > 3% → 实质性改善策略，保留
```

**跟动量因子的top3-bottom3 spread对标**：动量spread告诉你"能多赚多少"，Net_Benefit告诉你"能少亏多少"。一个策略的夏普改善 = 多赚的部分 + 少亏的部分 - 执行成本。两者评估的物理量的单位是一致的（%收益），可以直接加总。

#### 5.5 可视化诊断工具

**散点图 + LOESS拟合**——替代跑IC回归：

```
X轴：因子原始值（如 RV_20d_rank，从0到1）
Y轴：未来N日forward return
每点 = 一个交易日
叠加 LOESS 拟合线

理想形态：
  X在[0, 0.7]区间 → LOESS线接近水平（因子值低时没有预测力 = 不干扰正常交易）
  X在[0.7, 1.0]区间 → LOESS线明显向下弯曲（因子值高时return快速恶化）
  → 这说明因子只在尾部有预测力，正文区间不干扰

垃圾形态：
  LOESS线全程随机波动 → 无信息
  LOESS线全程向下倾斜 → 这其实是个alpha因子，不是风险因子（和动量重叠）
  LOESS线中间有奇怪的峰谷 → 过拟合/噪音
```

这比任何统计量都直观：你一眼就能看出因子在哪段工作、在哪段不工作。

**分布叠加图**：

```
把forward return分成两组画直方图：
  蓝色：Safe组（未触发）
  红色：Danger组（已触发）

理想形态：红色分布明显左偏且左尾更厚
垃圾形态：两坨分布几乎重叠
有害形态：红色分布右偏（触发后反而涨更多）
```

#### 5.6 与动量因子的评价体系对照

| | 动量因子（alpha） | 风险因子（断路器） |
|---|---|---|
| **本质** | 排序/选股 | 二元开关 |
| **核心评价量** | top3 - bottom3 spread (%) | RAS_bear / Profit_Sacrifice (%) |
| **正确的工作区间** | 全区间线性 | 仅尾部非线性 |
| **IC/rankIC** | 适用 | **不适用**（好的风险因子IC≈0） |
| **好因子的标志** | spread > 0，单调性好 | LOESS右端下弯 + 中部平坦 |
| **过拟合标志** | 参数多、spread高但不稳定 | FAR极低但CCR也低（阈值设太松） |
| **失效模式** | 因子拥挤导致spread衰减 | regime切换导致因子与历史尾部脱节 |
| **互补性要求** | 各因子信息源不同 | 与alpha因子信息源不同（相关<0.2） |

### 六、VIX专题

#### 6.1 VIX是什么

**VIX不是风险指标——VIX是波动率的市场价格。** 它不预测方向，它预测「未来30天SPX的波动幅度有多大」。但波动率上升几乎总是和下跌同步发生（非对称相关：跌1%→VIX涨4-5%，涨1%→VIX跌1-2%），所以VIX在实际使用中几乎等于恐慌指数。

```
VIX² = (2/T) × Σ (ΔK_i / K_i²) × e^{rT} × Q(K_i) - (1/T)(F/K_0 - 1)²

CBOE model-free公式。本质：把SPX所有OTM put和call价格加权求和，反推隐含波动率。
```

#### 6.2 VIX作为预测性因子的三个用法

**用法1：VIX水平 → regime判定**
```
VIX < 15 → 低波动regime，正常仓位
VIX 15-25 → 正常，不干预
VIX 25-30 → 高波动预警，仓位上限75%
VIX > 30 → 高波动regime，仓位上限50%
VIX > 40 → 危机regime，清仓
```

**用法2：VIX跳变 → 恐慌初现**
```
VIX_today / VIX_ma_10d > 1.2 → 恐慌正在蔓延 → 减仓至50%
```

**用法3：VIX期限结构 → 市场对恐慌持续性的定价**（最强信号）
```
Contango (VIX < M1 < M2) → 正常
Backwardation (VIX > M1 > M2) → 市场认为现在比未来更恐慌 → 恐慌可能见顶
曲线从contango翻转为backwardation → 减仓至25%或清仓
```

#### 6.3 VIX可以交易吗？

不能直接交易VIX现货。可以交易VIX期货、VIX期权、VIX ETN（VXX/UVXY/SVXY——但这些在contango中有巨大展期损耗，VXX历史上跌了99%+）。

**对你来说VIX最有用的不是交易它，而是作为regime输入。**

#### 6.4 A股VIX替代

- **iVIX**（上交所发布，基于50ETF期权链），Wind/Choice可获取
- 数据获取不便时可用的替代：50ETF已实现波动率分位数（因子1）+ 国债期货波动率（作为跨市场先行指标）

## 第二部分：韩国半导体做空——小本金路径

### 一、第一性原理层

你做的是两件事叠在一起：
1. **方向下注**：韩国半导体股价会跌
2. **时点下注**：这个下跌会在你的工具过期前发生

两件事同时对的概率 = P(方向对) × P(时点对|方向对)。如果方向判断到对的概率是60%，时点也对的概率是30%，那组合概率是18%。**这就是为什么大部分期权买方亏钱——不是方向错了，是时机错了。**

### 二、为什么韩国半导体？

当前背景（2026年7月）：韩国半导体疯涨，驱动是AI/HBM（高带宽内存）需求叙事——SK Hynix独供NVIDIA HBM3E，三星在追赶。

**潜在黑天鹅：**
- AI资本开支见顶 → HBM需求预期崩溃
- CXMT（长鑫存储）突破HBM技术 → 打破韩国垄断
- DRAM周期自然下行（存储是强周期品，历史上每3-4年一轮）
- 地缘风险（朝鲜/台海）
- 三星工艺良率问题（已不是第一次）

最大收益 = (K1 - K2) × 100 - 净成本
最大损失 = 净成本

优势：成本降低50-70%
劣势：封顶收益（如果EWY跌到$20，你也只能赚到$45为止）

### 四、小本金「黑天鹅下注」的数学

假设你有 10,000 RMB（约$1,400）的「赌博预算」：

**方案A：纯买Put（简单粗暴）**
```
单张 EWY Put 期权金 ≈ $200
持仓上限 ≈ 7张
EWY当前 ≈ $60

场景分析（6个月到期）：
┌─────────────────┬──────────┬────────────┬──────────┐
│ 6个月后 EWY价格   │ Put盈亏    │ 账户总盈亏   │ 发生概率   │
├─────────────────┼──────────┼────────────┼──────────┤
│ $75（继续涨25%）  │ -$1,400   │ -100%      │ 30%      │
│ $60（横盘）       │ -$1,400   │ -100%      │ 40%      │
│ $45（跌25%）      │ +$700     │ +50%       │ 20%      │
│ $30（跌50%）      │ +$5,600   │ +400%      │ 8%       │
│ $20（黑天鹅）     │ +$17,500  │ +1250%     │ 2%       │
└─────────────────┴──────────┴────────────┴──────────┘

期望值 = (-1400×0.70) + (700×0.20) + (5600×0.08) + (17500×0.02)
       = -980 + 140 + 448 + 350
       = -$42

期望值为负，这是你支付的权利金溢价。
```

**方案B：Put Spread（降低血耗）**
```
买 EWY $55 Put + 卖 $40 Put
净成本 ≈ $80/张（vs 纯买Put的$200）
持仓上限 ≈ 17张（$1,360）

最大收益封顶 = ($55-$40)×100 - $80 = $1,420/张
发生黑天鹅（EWY→$30）：收益封顶在$1,420/张，不如纯Put
但日常血耗大幅降低 → 你可以在赌桌上坐更久。
```

**方案C：Put Calendar Spread（时间结构博弈）**
```
卖近月OTM Put + 买远月OTM Put（同一行权价）
净成本更低，但是：如果暴跌发生在近月到期日之后，你只剩远月Put，vega暴露变大。

→ 复杂度高，不推荐小本金操作。
```

### 五、现实评估：这条路到底能不能走

**能走的部分：**
- EWY期权在美国市场，只要有美股期权交易权限就能操作
- 国内互联网券商（富途/老虎/盈透）都支持
- $1,000-$2,000就可以起步
- 韩国半导体确实存在周期性风险，不是纯粹的「赌」

**不能走/不该走的部分：**

1. **小本金+期权 = 极高概率归零。** 方向可能对，但时间不对，还是归零。你花了$1,400，有70%+概率6个月后归零。

2. **做空疯涨的东西 = 对抗momentum。** 韩国半导体现在是在momentum regime中，你进去做空本质上是「我认为现在就是顶」。这句话在每次牛市中被说过无数次，只有最后一次是对的。

3. **黑天鹅的概率被系统性高估。** 人脑对「可能发生的大灾难」天然给予过高权重。韩国半导体崩盘的概率是有的，但你能不能在它崩之前坐在赌桌上不破产？

4. **没有优势。** 你对韩国半导体的认知优于市场吗？你对HBM供需的预判比花旗/大摩的半导体分析师准吗？如果答案是否定的——你是在和市场对赌，而市场给你的赔率是由比你更了解韩国半导体的人定的。
