# 四象擒龙 ETF轮动稳健策略

基于动量因子的ETF轮动策略，结合多因子风控体系，追求稳健收益。

## 策略概述

本策略采用**动量选股 + 多重风控**的核心框架：

- **动量因子**：年化收益 × R²（对数收益率加权线性拟合）
- **风险控制**：波动率异常（标准差阈值）+ 流动性异常（变异系数阈值）+ 固定止损
- **调仓机制**：定期轮动（每N个交易日）+ 风险信号触发减仓

## 因子计算

### 1. 动量因子
```python
年化收益 × R²
```
- 回溯周期：20个交易日
- 计算方法：对数收益率加权线性回归（权重从1线性增至2）
- 物理含义：趋势强度 = 收益率 × 趋势稳定性

### 2. 风险因子
- **标准差因子**：`(20日收益标准差 + 5日收益标准差) / 2`
- **变异系数(CV)**：20日成交额标准差 / 20日成交额均值

## 交易逻辑

### 选股规则
- 每日收盘后计算所有ETF动量得分
- 选取动量排名前N名的ETF（默认N=4）
- T日收盘信号 → T+1日以OHLC4均价执行

### 调仓规则
- **定期调仓**：每N个交易日（默认10个交易日）
- **风险减半仓**：任一持仓ETF触发风险条件时，当日减半
  - 标准差 > 0.03 或 变异系数 > 0.5
- **新标的风险减半**：调仓买入时若触发风险条件，只买入一半金额

### 止损规则
- 单ETF从买入价下跌超过止损率（默认7%）时，次日止损出局

### 交易成本
- 佣金：万5（0.05%）
- 单笔最低：5元

## 使用方法

### 1. 配置环境
```bash
# 在 .env 文件中填入您的 Tushare Token
TUSHARE_TOKEN=your_token_here
```

### 2. 下载数据
```bash
python 1ETF数据下载.py
```

### 3. 计算因子
```bash
python 2动量指标计算.py
```

### 4. 运行回测
```bash
python 3ETF轮动策略_回测.py
```

### 5. 查看结果
回测结果保存在 `output/ETF轮动回测结果/` 目录下：
- `backtest_metrics.csv` - 年化收益率、夏普比率、最大回撤等核心指标
- `backtest_equity_curve.png` - 净值曲线可视化
- `rebalance_records.csv` - 每次调仓的详细记录

## 回测绩效指标

| 指标 | 说明 |
|------|------|
| 总收益率 | 期末净值 vs 期初净值 |
| 年化收益率 | 复利年化收益 |
| 年化波动率 | 日收益标准差 × √252 |
| 夏普比率 | 年化收益 / 年化波动率（无风险利率=0） |
| 最大回撤 | 历史最大浮亏 |
| 调仓次数 | 完整轮动次数 |
| 止损次数 | 触发止损的事件数 |
| 风险减半仓次数 | 波动率/流动性风险减仓次数 |

## 风险提示

1. **历史回测不代表未来表现**：策略过往业绩不应作为未来收益的保证
2. **参数过拟合风险**：建议使用样本外数据进行验证
3. **流动性风险**：部分ETF可能在极端行情下流动性不足
4. **市场环境变化**：动量策略在市场风格切换时可能失效
